# Watan Markt – Landing Page Redesign

## About
This project is a redesign of the Watan Markt landing page, completed as part of my WordPress Development Internship at Proton Soft Tech. Watan Markt is a South Asian/Middle Eastern grocery eCommerce website.

## Project Overview
The original landing page was audited for design flaws, and a new homepage was built with improved layout, visuals, and user experience.

## Features
- Hero section
- Category cards
- Best Sellers carousel
- Featured products section
- Trust badges
- Footer

## Tech Stack
- WordPress
- Elementor
- PHP
- CSS

## Setup
1. Clone the repository
2. Import the database
3. Configure `wp-config.php` with your database credentials
4. Run on localhost using Local by WP Engine
